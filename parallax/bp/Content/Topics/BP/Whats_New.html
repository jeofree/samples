Whats New in Brand Protection

Agari is always working to improve the Brand Protection product, from fixing issues to improving existing features to adding new features. This topic highlights the feature changes in Brand Protection, as well as documentation updates not necessarily related to product features.

Release Date Update Details
2020.12 December 2020 Minor edits
   

  • Note: To better align guide release names and dates, guide versions published at the end of the month will now be named for the upcoming month instead of the previous month. Because of this there is no 2020.11 version of this guide.

2010.10 October 2020 Misc edits and fixes
2020.09 September 2020 Misc edits and minor updates
2020.08 August 2020
  • Added Inbound DMARC Visibility, see How DMARC Works

 

  • Added an update describing accessing the Threat Feed whitelist page from the Configure menu. See Access the Whitelist from the Configuration Menu.
2020.04 April 2020 A new Executive Reports page provides at-a-glance reports for several key facets of your domain infrastructure. See Executive Overview.
2020.03 March 2020

This update includes just some minor documentation bug fixes.
2020.02 February 2020

This update includes the following improvements to the Brand Protection documentation:

  • Corrections were made to the roles that can be assigned to users. One role was renamed, and a new role was added and documented. See User Account Settings.
  • The Brand Protection Threat Feed functionality is now documented. See. Threat Feed.
  • How you can search the Brand Protection audit trail is now documented. See Search Organization Activity.
2019.12 December 2019 Product style sheets (CSS) incorporated some fixes as a part of an infrastructure upgrade, so dialog box title text now contrasts with the background and is easier to read. Screen shots in this document have been updated.
2019.11 November 2019

This November documentation release focuses on improving its content, including:

  • Reorganized this guide. High-level sections now include subjects such as:
    • Email standards
    • Getting to reject (DMARC)
    • Monitoring your outgoing email ecosystem once your get to reject
  • Added a topic on understanding identifier alignment. See Understanding Identifier Alignment.
  • Added a topic on how to whitelist alerts. See Whitelist Alerts.
  • Added a topic on SPF record length. See SPF Record Length.
  • Added information about the Threat Feed. See Threat Feed.
2019.10 October 2019
  • Clarified that Agari employees cannot make changes to Brand Protection user accounts. See User Accounts.
2019.04 April 2019
  • DKIM Management and Hosting

Agari added the capability to host DKIM records. Brand Protection can now manage DKIM records hosted by Agari. The DKIM management in Brand Protection can discover existing DKIM keys used for domains. Agari can now host all DNS records relating to DMARC, SPF, and DKIM. Read more at Host Your DKIM Records at Agari.
