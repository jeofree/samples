Creating link renderers

In the same way as for node renderers, you will need to create your own graphical representation and indicate to the Diagram component to create your link renderer instances instead of the default ones.
This configuration can be achieved using the following properties:
  • linkRenderer
  • linkRendererFunction
ThelinkRenderer and linkRendererFunction properties work in the same way as the node renderer properties when creating links. Once a renderer has been created to represent an object from the data provider, it can be retrieved from the Diagram using thegetRenderer function, as in the following example: 
var r:Renderer = diagram.getRenderer(item);
