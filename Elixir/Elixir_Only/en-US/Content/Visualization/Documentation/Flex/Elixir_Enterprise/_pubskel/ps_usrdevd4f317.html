Basic representation of nodes and links

Objects that are part of the Diagram 'data providers' are graphically represented in the diagram by renderer instances. By default, nodes are represented as instances of the Node class, whereas links are represented as instances of the Link class. These renderers provide a basic representation, which shows the relationships between objects and a label on each node, as illustrated in the following figure.
