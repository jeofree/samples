Introduction to Gauges

IBM® ILOG Elixir Gauges help improve data visualization applications for analysis and real-time monitoring of data. Gauges include predefined circular and rectangular (both horizontal and vertical) gauges, as well as a dedicated framework to construct custom gauges. ILOG Elixir Spark-based Gauges leverage Flex 4 component architecture, based on a strong separation of the logic and the appearance of user interface components. This means that you can customize every aspect of the appearance of a gauge by modifying its skin
Gauge components are interactive. You can edit the data values displayed by the gauges using the mouse or the keyboard. Gauges also provide customizable animations on data changes. IBM® ILOG Elixir provides both predefined gauges and a framework for building custom gauges. If you need to display a single value on a single scale, use or customize the predefined gauges. If you need to display several values or scales, use the Gauge framework to build your own custom gauges.
Note
This gauge module based on Spark technology replaces the previous gauge module available inIBM® ILOG Elixir since version 1.0. For instructions on how to enhance gauges created with earlier versions of Elixir using MX gauge technology, see:MX Gauges in the Appendix: MX gauges and maps
