Creating a map control

The Maps control displays a set of map features. Map features are cartographic entities that represent geographic entities such as a country or a state. The Maps control can only display area type map features. Map features can be styled and customized through the data provider set on the Maps control.
The Maps control is a subclass of the component SkinnableComponent. This means that you can apply a skin to it, and can also define a custom item renderer for it. For more information on the SkinnableComponent, see the components Group and Skin-nableComponent.
Using the Map control styles
The Maps control defines several styles that you can use to modify the basic appearance of the control. You can use styles to make some visual changes to the control, but the best way to have total control over the display is to create a skin.
The following example customizes the map using the contentBackgroundColor, content-StrokeColor, selectionColor, rollOverColor and background style properties. 
<ibm:WorldCountriesMap selectionMode="multiple"
                                            height="400" width="800"
                                            contentBackgroundColor="red"
                                            contentStrokeColor="green"
                                            selectionColor="blue" 
                                            rollOverColor="cyan" 
                                            backgroundColor="yellow" />
