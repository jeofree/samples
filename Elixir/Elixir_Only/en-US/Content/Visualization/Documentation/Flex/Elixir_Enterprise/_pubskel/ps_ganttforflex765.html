Link properties

The color and thickness of links that connect the items can be configured by setting one or several style properties depending on the desired granularity.
The linkColor and linkThickness style properties allow you to specify the color and thickness respectively of all the links in the chart.
The linkStroke style property allows you to specify all the properties of the stroke used to draw the links in the chart. The linkStrokeFunction property gives a finer granularity by allowing you to specify the stroke to use to draw the links between a given item and its children. The linkStrokeInherited property can be used with a custom linkStrokeFunction to make another link inherit the custom stroke or not.
To determine the stroke to use, the OrgChart component proceeds as follows:
  1. If linkStrokeFunction is defined, it retrieves the stroke associated with the current item.
  2. If there is no stroke associated with the current item (result null) but linkStrokeInherited is set to true, it finds the nearest parent in the hierarchy that has a stroke defined and uses it.
  3. If there is no stroke associated with the current item (result null) and linkStrokeInherited is set to false or if linkStrokeFunction is not defined, it uses the linkStroke style property.
  4. If the linkStroke property is not set, it builds the stroke using the linkColor and linkThickness style properties.
Example 1
All links are set green with a thickness of 3 pixels at identity using the linkColor and linkThickness properties.
<?xml version="1.0" encoding="utf-8"?>
<mx:Application xmlns:mx="http://www.adobe.com/2006/mxml" 
                xmlns:ilog="http://www.ilog.com/2007/ilog/flex">
  <ilog:OrgChart width="100%" height="100%"
                 linkColor="#00FF00" linkThickness="3">
    <ilog:dataProvider>
      <mx:XML>
        <person name="person 1">
          <person name="person 2">
            <person name="person 3"/>
            <person name="person 4"/>
          </person>
          <person name="person 5">
            <person name="person 6"/>
            <person name="person 7"/>
          </person>
        </person>        
      </mx:XML>
    </ilog:dataProvider>
  </ilog:OrgChart>
  
</mx:Application>
Example 2
Same as Example 1, but with no rescale on zooming and link set to semitransparent using the linkStroke style property.
<?xml version="1.0" encoding="utf-8"?>
<mx:Application xmlns:mx="http://www.adobe.com/2006/mxml" 
                xmlns:ilog="http://www.ilog.com/2007/ilog/flex">
  <ilog:OrgChart width="100%" height="100%">
    <ilog:linkStroke>
      <mx:Stroke alpha="0.5" color="#00FF00" weight="3" scaleMode="none" />
    </ilog:linkStroke>
    <ilog:dataProvider>
      <mx:XML>
        <person name="person 1">
          <person name="person 2">
            <person name="person 3"/>
            <person name="person 4"/>
          </person>
          <person name="person 5">
            <person name="person 6"/>
            <person name="person 7"/>
          </person>
        </person>        
      </mx:XML>
    </ilog:dataProvider>
  </ilog:OrgChart>
  
</mx:Application>
Example 3
Link color is set as specified in a property of a data item with no inheritance using the linkStrokeFunction style property.
<?xml version="1.0" encoding="utf-8"?>
<mx:Application xmlns:mx="http://www.adobe.com/2006/mxml" 
                xmlns:ilog="http://www.ilog.com/2007/ilog/flex">
  <mx:Script>
    <![CDATA[
      import mx.graphics.Stroke;
      import mx.graphics.IStroke;
      
      private function linkStrokeFunction(item:XML):IStroke {
        if (item.hasOwnProperty("@linkColor")) {
          return new Stroke(StyleManager.getColorName(item.@linkColor.toString()), 3);
        } 
        
        return null;       
      }
    ]]>
  </mx:Script>
  <ilog:OrgChart width="100%" height="100%"
                 linkStrokeFunction="{linkStrokeFunction}"
                 linkStrokeInherited="false">
    <ilog:dataProvider>
      <mx:XML>
        <person name="person 1" linkColor="#00FF00">
          <person name="person 2">
            <person name="person 3"/>
            <person name="person 4"/>
          </person>
          <person name="person 5" linkColor="#FF0000">
            <person name="person 6"/>
            <person name="person 7"/>
          </person>
        </person>        
      </mx:XML>
    </ilog:dataProvider>
  </ilog:OrgChart>
  
</mx:Application>
