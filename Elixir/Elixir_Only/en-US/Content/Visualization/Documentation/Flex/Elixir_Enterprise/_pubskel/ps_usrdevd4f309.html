Creating a diagram with nodes and links

The Diagram class is a Flex® UIComponent that automatically creates a graphic representation of a graph composed of nodes and links in a business data model. Like many other Flex graphic components, it relies on the Flex “data provider” paradigm. In the case of the Diagram class, two properties act as data providers: nodeDataProvider for nodes and linkDataProvider for links. The Diagram class takes care of rendering the nodes and links found in the nodeDataProvider and linkDataProvider properties, and listens for changes in these data sources to reflect them in the graphical nodes and links of the diagram.
The following figure shows how to display a basic graph using a Diagram component.
<?xml version="1.0" encoding="utf-8"?>
<s:Application xmlns:fx="http://ns.adobe.com/mxml/2009" 
               xmlns:s="library://ns.adobe.com/flex/spark"
               xmlns:ibm="http://www.ibm.com/xmlns/prod/ilog/flex/diagram/2010" 
               xmlns:mx="library://ns.adobe.com/flex/halo"
               creationComplete="fillDataProviders()">
  <fx:Script>
    <![CDATA[
      import mx.collections.ArrayCollection;
      
      [Bindable]
      public var nodes:ArrayCollection;
      [Bindable]
      public var links:ArrayCollection;
      
      public function fillDataProviders():void {
        nodes = new ArrayCollection();
        var n:Object = new Object();
        n.label="Node 1";
        n.x = 10;
        n.y = 10;
        nodes.addItem(n);
        n = new Object();
        n.label = "Node 2";
        n.x = 70;
        n.y = 200;
        nodes.addItem(n);
        n = new Object();
        n.label = "Node 3";
        n.x = 150;
        n.y = 50;
        nodes.addItem(n);
        
        links = new ArrayCollection();
        var link:Object = new Object();
        link.start = nodes[0];
        link.end = nodes[1];
        links.addItem(link);
        link = new Object();
        link.start = nodes[0];
        link.end = nodes[2];
        links.addItem(link);
        link = new Object();
        link.start = nodes[1];
        link.end = nodes[2];
        links.addItem(link);
      }
    ]]>
  </fx:Script>
  
  <ibm:Diagram id="diagram" left="0" top="0" width="100%" height="100%" 
               nodeDataProvider="{nodes}" linkDataProvider="{links}"
               xLocationField="x" yLocationField="y"
               startNodeField="start" endNodeField="end"
               labelField="label">
  </ibm:Diagram>

</s:Application>
