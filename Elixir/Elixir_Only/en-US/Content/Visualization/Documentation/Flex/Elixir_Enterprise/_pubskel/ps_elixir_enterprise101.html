Accessibility in gauges

The GaugeBase component and all its subclasses including predefined gauges have built-in support for accessibility: This automates the common accessibility practices of providing text equivalents and promoting keyboard access. The accessibility implementation class of a gauge component is GaugeBaseAccImpl. The GaugeBase component is revealed through the slider role of MSAA (Microsoft® Active Accessibility) and provides as value the value of the gauge. You can customize the gauge value label provided to the screen reader by setting a custom label function in the labelFormatFunction property of the GaugeMarkerBase class, or by redefining the formatValue method of the GaugeBaseAccImpl class. The user of the gauge can use the arrow keys to navigate through the various gauge values or can change the gauge values as described in XXXX (see with DamienM for the title of this section). For more information on accessibility, see Creating Accessible Applications in the Advanced Flex ProgrammingAdobe documentation.
