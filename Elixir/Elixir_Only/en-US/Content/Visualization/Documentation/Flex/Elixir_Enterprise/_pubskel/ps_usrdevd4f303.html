Learning about IBM Diagram

To learn about IBM® Diagram quickly:
  1. Read the Overview of IBM Diagram.
  2. Scan the Tutorials, beginning with Add tutorial name.
  3. Identify the samples that are relevant to your development needs by using the Sample Browser (Start Menu>IBM ILOG>D4F>Samples), which allows you to scan what the product can offer you. The samples are installed in <installdir> /samples. Use the samples as a foundation for developing your own IBM Diagram applications.
  4. As you begin to develop your applications, continue to use the documentation as a primary source of information. The documentation is compiled into an easy-to-use Help file that is displayed in the Adobe® Flash® Builder™ 4 Help window.
