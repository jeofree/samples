Map control user interaction

Selection
Map features can be selected or not depending on the value of the selectionMode property of the map. If this property is set to “none”, no selection is possible. If set to “singleSelection”, only one feature can be selected at a time. If set to “multipleSelection”, the selection can be adjusted using a keyboard modifier. This keyboard modifier is hosted by the adjustSelectionModifier property.
Selected item indices can be retrieved in the selectedIndices property. The selected items can be retrieved in the selectedItems property of the map. The selectedIndices is a Vector of type int, and the selectedItems property is a Vector of type MapFeature
The following example shows the three selection modes in action:
<?xml version="1.0" encoding="utf-8"?>
<s:Application xmlns:fx="http://ns.adobe.com/mxml/2009"
               xmlns:s="library://ns.adobe.com/flex/spark"
               xmlns:mx="library://ns.adobe.com/flex/halo"
               xmlns:ibm="http://www.ibm.com/xmlns/prod/ilog/elixir/2010">
  <mx:HDividedBox width="100%" height="100%">
    <s:VGroup width="100%" height="100%">
      <ibm:ContinentalEuropeMap id="map" width="100%" height="100%"
        mapChange="area.text = 'indices : ' + map.selectedIndices + '\n' + map.selectedItems"/>
      <s:HGroup>
        <s:RadioButton id="none" label="none"
             selected="true"
             change="if(none.selected) map.selectionMode = 'none'"/>
        <s:RadioButton id="single" label="single"
             change="if(single.selected) map.selectionMode = 'single'"/>
        <s:RadioButton id="multiple" label="multiple"
             change="if(multiple.selected) map.selectionMode = 'multiple'"/>
      </s:HGroup>
    </s:VGroup>
    <s:TextArea id="area" width="100%" height="100%"/>
  </mx:HDividedBox>
</s:Application>
Scroller
When associated with a Scroller, the map component is able to navigate through the map. Keyboard modifiers and mouse button associations for panning and zooming are defined in the Scroller. To pan through or zoom into the map, use ._____ By default the Map-Skin skin uses a scroller. See ‘Using Maps control skins’ for more information about using a Scroller control with the Maps component.
Keyboard navigation
Keyboard navigation is also possible in the map component, available if the selection-Mode property is set to “single”. In this case, the keyboard navigation is performed by pressing the keyboard modifier and the arrow keys that are defined for navigation. The keyboard modifier is defined by the keyboardNavigation property of the map.
