Your IBM ILOG ElixirDiagram installation

After installation, IBM® ILOG ElixirDiagram is typically in one of the following directories:
  • On Microsoft® Windows® the installation directory is normally: c:\Program Files\IBM\ILOG\Elixir 3.0
  • On Mac OS® the installation directory is normally: /Applications/IBM/ILOG/Elixir 3.0
  • On Linux® , the installation directory is normally:
    ~/IBM/ILOG/Elixir 3.0, for example:
    /usr/julian/IBM/ILOG/Elixir 3.0
  • On Microsoft® Windows® the installation directory is normally: c:\Program Files\IBM\ILOG\D4F 3.0
  • On Mac OS® the installation directory is normally: /Applications/IBM/ILOG/D4F 3.0
  • On Linux® , the installation directory is normally:
    ~/IBM/ILOG/D4F 3.0, for example:
    /usr/julian/IBM/ILOG/D4F 3.0
The installation directory contains the IBM ILOG ElixirDiagram libraries (SWC and SWF files) and IBM ILOG ElixirDiagram samples.
It also contains the Custom Map Converter utility.
According to the information you gave during the IBM ILOG ElixirDiagram installation process, the installation process for Eclipse® or Adobe Flash Builder 4 also installs documentation. The documentation is available through the standard Eclipse Help menu or the Help menu in the builder.
You can run the samples from the Start>IBM ILOG>ILOG ElixirDiagram menu (for Microsoft® Windows® ), from the <installation-dir>/samples directory (for Mac OS® ), or from the <installation-dir>/samples directory (for Linux® ). Each sample contains its source code and a build executable that allows you to recompile the sample if necessary.
