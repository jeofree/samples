Managing selected objects

D4F provides an API to manage the selection of objects in a diagram.
This API is composed by the properties and methods available in the Diagram class:
  • allowSelection
  • selectionMode
  • selectedItems
  • selectedItem
  • setItemSelected
  • isItemSelected
  • clearSelection
The current selection is available in the selectedItems property. The selectedItem property returns the first selected item in the list. The elements part of current selection are the business data objects present in the diagram data providers. You can modify the list of selected items using methods setItemSelected and clearSelection or modifying the value of property selectedItems. The selected items must be the business data objects present in the diagram data providers. You can query whether an item is selected using method isItemSelected. You can also forbid any kind of selection by setting property allowSelection to false. By default, this property is true, which means that objects can be selected. Selection behavior is specified by property selectionMode.
The possible configuration is:
  • Single selection
  • Multiple selection
  • Extended selection
In addition to the API, you can also manage the selection by interacting directly in the Diagram. See details in LINKTO(User interaction with Diagrams).
