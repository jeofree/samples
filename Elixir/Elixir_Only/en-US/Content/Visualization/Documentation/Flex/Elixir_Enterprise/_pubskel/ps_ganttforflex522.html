Common properties of Gantt sheet grids

A grid displayed by the Gantt sheet is called a Gantt sheet grid. Gantt sheet grids share properties through their common superclass, the GanttSheetGridBase class

Display Mode of a grid

A Gantt sheet grid can be displayed in two ways:
  • On the whole Gantt sheet
  • On a per-row basis
To specify which mode to use, set the drawPerRow property in the GanttSheetGridBase class. When the grid is drawn on the whole Gantt sheet (drawPerRow is set to false), the grid is displayed in one operation, on the whole Gantt sheet area. When the grid is drawn on a per-row basis, the grid is displayed independently in each row, making it possible to display different things in each row. For example, if you want to have the same background color for all the Gantt sheet rows, you can use a BackgroundGrid with the drawPerRow property set to false. If you want to have different background colors in each row, set the drawPerRow property to true. The default value for this property depends on each grid implementation.
When a grid is drawn on a per-row basis, the developer has to provide a way for the grid to retrieve the data that can be used to draw each row. For example, if you want to display a specific background color for each Gantt sheet row using a BackgroundGrid, you have to specify how to get the color from the data displayed in the row. For the RowGrid class, you can use the backgroundField property. Each Gantt sheet grid implementation provides its own set of properties that can be use to map data properties to grid drawing properties. For details, refer to the documentation of each Gantt sheet grid class.

Display Area of a Grid

A Gantt sheet grid can be displayed on the whole Gantt sheet area, or only on the area where valid rows are displayed. You can control this behavior by using the GanttSheetGridBase.drawToBottom property in the GanttSheetGridBase class. The default value for this property is true
