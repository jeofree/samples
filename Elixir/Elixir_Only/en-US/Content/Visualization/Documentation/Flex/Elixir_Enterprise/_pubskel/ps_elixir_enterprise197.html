Managing radar chart events

The radar chart control manages user interaction events. The radar chart component exposes all the Adobe® Flash® Builder™ 4 events, but does not extend them. The available events are described in About charting events in the Flex 4 Developer’s Guide.
The following code shows how to listen for an itemClick event on chart columns that calls an itemClickAction function:
<ilog:RadarChart height="100%" width="100%"
  itemClick="itemClickAction()"
  dataProvider="{stockDataAC}">
