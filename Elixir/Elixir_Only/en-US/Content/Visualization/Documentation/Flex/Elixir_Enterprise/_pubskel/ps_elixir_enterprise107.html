Accessibility in indicators

The IndicatorBase component and all its subclasses AlphaNumericIndicator14, AlphaNumericIndicator35 and NumericIndicator, have built-in support for accessibility. This automates the common accessibility practice of providing a text equivalent to the indicator. The accessibility implementation class of an indicator component is IndicatorBaseAccImpl. The IndicatorBase component is revealed through the static text role of MSAA (Microsoft® Active Accessibility) and provides the text of the indicator as its name. Because the indicator component is not focusable in JAWS, its description will only be available in Virtual PC Cursor mode. For more information on accessibility, see Creating Accessible Applications in the Advanced Flex Programming Adobe documentation.
