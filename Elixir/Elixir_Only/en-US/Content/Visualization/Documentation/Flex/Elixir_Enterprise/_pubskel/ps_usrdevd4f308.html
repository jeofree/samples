Renderer

In IBM® Diagram, renderers are used to represent each node or link displayed in a graph. A renderer is an instance of the Renderer class.
Renderers are skinnable components that inherit from the Adobe® Flex® SkinnableComponent class. IBM Diagram provides renderers to represent nodes, links and subgraphs. As a skinnable component, the renderer class defines the core behavior of the nodes, links, and subgraphs. This behavior includes the events dispatched by the component, the data that the component represents, the skin parts that should be implemented by the skin class, and the view states that the skin class should support. The skin class manages the appearance of the component. IBM Diagram provides default skins for each of the renderers. You can create your own graphic representation by creating skins according to your needs and associating them with the renderers. In IBM Diagram, nodes are implemented by the Node class and links between nodes are implemented by the Link class. In conjunction with the Link class, the Port class allows you to specify how a link connects to a node. For details, see Introducing Link and Port Classes.
