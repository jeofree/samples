Customizing nodes, links and subgraphs in a diagram

Describes the representation of nodes, links, and subgraphs by renderers and how to customize it.
In this section
  • Basic representation of nodes and links
  • Creating node renderers
    • Explains how to use the renderer function to specify the node classes that you want to use.
  • Creating link renderers
    • Explains how to use the renderer function to specify the link classes that you want to use.
  • Creating subgraph renderers
    • Explains how to use the renderer function to specify the subgraph classes that you want to use.
  • Customizing node and link renderer skins
    • Explains how you can change node, link and subgraph renderer skins and how the skins may be directly linked to data in the data source.
  • Configuring node, link, and subgraph renderers
    • Explains how you can tweak the generated node, link, and subgraph by code when events are fired.
