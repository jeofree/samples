Data tips

The OrgChart item renderers (default and detailed) have two data tips, one called just “data tip” and one called “presence indicator tip”.

The data tip

When the item renderer is at the first level of detail, a data tip is displayed when the user of the chart hovers over an item renderer. To prevent the display of this data tip, set the showDataTip style property to false.
By default, the data tip shows the name of the displayed data item. To customize the data tip, set the dataTipFunction property of the OrgChart component.

The presence indicator tip

When the user hovers over the presence indicator of the default item renderer, a tip is displayed to describe the indicator color. To prevent the display of this tip, set the showPresenceIndicatorTip style property to false.
By default, the presence indicator tip shows a set of predefined localized values. To customize the presence indicator tip, set the presenceIndicatorTipFunction property of the OrgChart.
