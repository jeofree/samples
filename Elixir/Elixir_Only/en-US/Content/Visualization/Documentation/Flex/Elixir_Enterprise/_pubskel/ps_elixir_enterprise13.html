Building an IBM ILOG ElixirDiagram application using Adobe Flash Builder 4

Setting up the application project

You can use the IBM® ILOG ElixirDiagram components in an Adobe® Flex® application alongside Adobe Flex components.
For more information about building your application using Adobe Flash Builder 4, see Building Projects in the Using Flex Builder 4 Adobe documentation.
To set up the application project for your IBM ILOG ElixirDiagram application:
  1. In Adobe Flash Builder 4, create a new Adobe Flex project, either for the Web (run in Flash® Player™ ), or for the Desktop (run in AIR™ ), by clicking File>New.
  2. Click the right mouse button on your project and choose the Properties menu item.
  3. Choose the Flex Build Path or AIR Build Path section accordingly.
  4. In the library path tab, add the IBM ILOG ElixirDiagram SWC files to your project using the Add SWC… button.
    IBM ILOG ElixirDiagram comes in the form of two SWC files:
    • The file ilog-elixir.swc contains the IBM ILOG ElixirDiagram classes and can be found in the <installation-dir>/frameworks/libs directory.
    • The file ilog-elixir_rb.swc contains US resource bundles for IBM ILOG ElixirDiagram and can be found in the <installation-dir>/frameworks/locale/en_US directory.
  5. Optionally, you can indicate to Adobe Flash Builder 4 where it can find the IBM ILOG ElixirDiagram source code to help with debugging.
    In the Flex Build Path or AIR Build Path section:
    • Expand the ilog-elixir.swc tree item and edit the Source attachment item.
    • Specify the path to the source: <installation-dir>/frameworks/projects/ilog-elixir/src.
For details of the installation directories for Microsoft® Windows® , Mac OS® , and Linux® , see Your IBM ILOG ElixirDiagram installation.
When you have created your application files, you can then do one of the following:
  • Instantiate them in ActionScript® files (.as):
    Example: var myTreeMap:TreeMap = new TreeMap();
  • Use them in MXML files (.mxml):
    Example: <ilog:TreeMap id="myTreeMap"/>
  • Use them in a design view:
    You can use the drag-and-drop feature to move components from the IBM ILOG ElixirDiagram folder of the component panel onto the design view.

Compiling and running the application

You compile and run the application from the main MXML file.
To compile and run the application:
  • Click the right mouse button on the main MXML file of your application and choose the Run Application menu item to get your IBM ILOG ElixirDiagram application compiled and running.
