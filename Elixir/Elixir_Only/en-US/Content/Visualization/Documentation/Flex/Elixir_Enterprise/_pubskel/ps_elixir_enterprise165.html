Setting the item renderer

The following example shows how to set an item renderer in MXML.
<?xml version="1.0" encoding="utf-8"?>
<mx:Application xmlns:mx="http://www.adobe.com/2006/mxml" layout="absolute"
 xmlns:ilog="http://www.ilog.com/2007/ilog/flex">
  <ilog:OrgChart width="400" height="200" >
    <ilog:itemRenderer>
      <mx:Component>
        <ilog:OrgChartDetailedItemRenderer/>
      </mx:Component>
    </ilog:itemRenderer>
  </ilog:OrgChart>
</mx:Application>
For better text rendering, especially during animations, use embedded fonts in the item renderers. The following example shows the use of an embedded font.
<?xml version="1.0" encoding="utf-8"?>
<mx:Application xmlns:mx="http://www.adobe.com/2006/mxml" 
        xmlns:ilog="http://www.ilog.com/2007/ilog/flex">
  <mx:Style>
    @font-face {
      fontFamily: MyFont;
      src: local("sansserif");
      fontWeight: normal;    
      embedAsCFF: false;      
    }
    </mx:Style>
  
  <ilog:OrgChart width="100%" height="100%">
    <ilog:dataProvider>
      <mx:XML>
        <p name="Person 1">
          <p name="Person 2"/>
          <p name="Person 2"/>
        </p>
      </mx:XML>      
    </ilog:dataProvider>
    <ilog:itemRenderer>
      <mx:Component>
        <ilog:OrgChartItemRenderer fontFamily="MyFont" />
      </mx:Component>
    </ilog:itemRenderer>
  </ilog:OrgChart>
</mx:Application>
For more information on embedded fonts, see Using fonts>Using embedded fonts in the User interfaces section of the Adobe documentation.
