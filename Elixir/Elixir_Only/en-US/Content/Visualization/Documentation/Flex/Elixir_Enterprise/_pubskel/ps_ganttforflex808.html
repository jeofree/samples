Accessibility in treemap controls

The TreeMap component has built-in support for accessibility: it automates the common accessibility practices of providing text equivalents and promoting keyboard access.
The accessibility implementation class of a TreeMap component is TreeMapAccImpl.
The TreeMap component is exposed through an outline role of MSAA (Microsoft® Active Accessibility).
The label of an event exposed to a screen reader is accessible through the itemToLabel() method. By default, this method returns the value of the name field of the data item. You can customize the label by providing a custom label function in the labelFunction property.
The user of the chart can use the arrow keys to navigate through displayed items with the keyboard as described in User interaction with treemaps.
For more information on accessibility, see Creating Accessible Applications in the Advanced Flex Programming> Adobe documentation.
