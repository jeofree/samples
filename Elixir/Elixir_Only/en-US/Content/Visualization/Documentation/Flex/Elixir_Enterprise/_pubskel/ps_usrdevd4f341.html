Creating graph displays by specifying nodes and links

The Diagram component (as explained in Creating a graph display from data providers using the Diagram class) creates a graph from data providers and generates node and link instances of the Node and the Link classes to render the data inside the data source. However, you can also create a graph by creating instances of the Node and Link classes and using them directly inside a Graph component. The Graph component is a group that is able to display nodes and links. You can specify the position of nodes through the top or verticalCenter and left or horizontalCenter properties of the elements. You can also lay out the graph using graph layout algorithms that you specify through the nodeLayout and linkLayout graph properties. Layout algorithms can be automatically executed as nodes or links are added or removed, or any changes are made to the structure of the graph, or upon request, according to application needs. The following example illustrates a Graph where nodes and links are created and placed using a graph layout algorithm:
<?xml version="1.0" encoding="utf-8"?>
<s:Application xmlns:fx="http://ns.adobe.com/mxml/2009" 
               xmlns:d="http://www.ibm.com/xmlns/prod/ilog/flex/diagram/2010" 
               xmlns:s="library://ns.adobe.com/flex/spark" 
               minWidth="1024" minHeight="768">
  <s:layout>
    <s:VerticalLayout paddingBottom="20" paddingLeft="20" paddingRight="20" paddingTop="20"/>
  </s:layout>
  <d:Graph automaticGraphLayout="true">
    <d:Node id="node1" label="node 1"/>
    <d:Node id="node2" label="node 2"/>
    <d:Node id="node3" label="node 3"/>
    <d:Node id="node4" label="node 4"/>
    <d:Node id="node5" label="node 5"/>
    <d:Node id="node6" label="node 6"/>
    
    <d:Link startNode="{node1}" endNode="{node3}"/>
    <d:Link startNode="{node1}" endNode="{node2}"/>
    <d:Link startNode="{node1}" endNode="{node4}"/>
    <d:Link startNode="{node4}" endNode="{node5}"/>
    <d:Link startNode="{node4}" endNode="{node6}"/>
    <d:nodeLayout>
      <d:TreeLayout/>
    </d:nodeLayout>
  </d:Graph>
</s:Application>
