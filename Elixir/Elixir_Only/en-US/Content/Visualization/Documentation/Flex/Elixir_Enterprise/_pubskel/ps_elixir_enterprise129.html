Using the MapDataLayer

The MapDataLayer is a group that uses the dataProvider defined on the MapBase component to potentially create a DataRenderer for each item found in the dataProvider.
The following example creates a Label displaying the name of a capital and placed at the location specified in the dataProvider:
Main application and data provider definition:
  <fx:Declarations>
    <s:ArrayList id="locations">
      <fx:Array>
        <fx:Object name="Paris" latitude="48.85341" longitude="2.3488"
                   country="FRA"/>
        <fx:Object name="London" latitude="51.5084152563931"
                   longitude="-0.125532746315002" country="GBR"/>
        <fx:Object name="Madrid" latitude="40.4165020941502"
                   longitude="-3.70256423950195" country="ESP"/>
        <fx:Object name="Berlin" latitude="52.5166667" longitude="13.4"
                   country="DEU"/>
        <fx:Object name="Roma" latitude="41.9" longitude="12.4833333"
                   country="ITA"/>
        <fx:Object name="Brussels" latitude="50.8465974826927"
                   longitude="4.35277462005615" country="BEL"/>
      </fx:Array>
    </s:ArrayList>
  </fx:Declarations>
  <ibm:ContinentalEuropeMap width="100%" height="100%"
                            dataProvider="{locations}"
                            skinClass="EuropeDataSkin"/>
The EuropeDataSkin defines a MapDataGroup referencing a EuropeSymbolRenderer. The keyField property is used to map the data item with the …. 
<ibm:Scroller panModifierKey="shift" wheelZoomModifierKey="shift" top="0"
                left="0" width="100%" height="100%" id="scroller"
                skinClass="com.ibm.ilog.elixir.skins.spark.ScrollerSkin">
    <ibm:MapGroup id="mapGroup">
      <ibm:MapFeatureGroup id="featureGroup" 
            featureRenderer="com.ibm.ilog.elixir.maps.skins.DefaultFeatureRenderer"/>
      <ibm:MapDataGroup itemRenderer="EuropeSymbolRenderer" zoomable="false"
                                          keyField="country"/>
    </ibm:MapGroup>
  </ibm:Scroller>
The EuropeSymbolRenderer creates a Label and places it at the longitude and latitude found in the data item. The DataRenderer uses verticalCenter and horizontalCenter to place the Label. Top, left, right or bottom properties could also have been used to accomplish this.
<?xml version="1.0" encoding="utf-8"?>
<s:DataRenderer xmlns="http://ns.adobe.com/mxml/2009"							    xmlns:s="library://ns.adobe.com/flex/spark				         
                            xmlns:ibm=http://www.ibm.com/xmlns/prod/ilog/elixir/2010
                            xmlns:mx="library://ns.adobe.com/flex/halo"						   verticalCenter="{data.latitude}"
                           horizontalCenter="{data.longitude}">
	<s:Label text="{data.name}" color="red"/>
</s:DataRenderer>
