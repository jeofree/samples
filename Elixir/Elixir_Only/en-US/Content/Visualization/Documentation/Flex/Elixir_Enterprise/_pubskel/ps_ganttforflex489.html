Accessibility

To accelerate the building of accessible applications, IBM® ILOG Elixir has built-in support for accessibility: the common accessibility practices of providing text equivalents, labeling controls, and promoting keyboard access are automated.
IBM ILOG Elixir comes with the following set of accessible components and related information.
Component Accessibility information
3D Charts Accessibility in 3D chart components
Calendar Accessibility in calendars
Organization Chart Accessibility for organization charts
Radar Chart Accessibility in radar charts
Treemap Accessibility in treemap controls
To enable accessibility in your application, check the Generate accessible SWF file option in the Flex Library Compiler section of the Adobe® Flash® Builder™ 4 project properties.
If you use the JAWS® (Job Access With Speech) screen reader, configure it in Forms mode to use the documented keyboard shorcuts.
